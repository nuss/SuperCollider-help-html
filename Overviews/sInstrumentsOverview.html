        sInstrumentsOverview
        Extension

        overview of instruments and classes

        Description

        Instruments developed in collaboration with Tamas Ungvary.

        Work in progress.

        Instruments

        These are complete instruments and can be found in the folder instruments.

        sGliss
        a glissing sInstrument.
        sDrone
        a droning sInstrument based on the Ndef used for 'Strange Foreign Bodies'.
        sFormant
        a formant sInstrument based on optoformant3.
        sCluster
        a cluster sInstrument.
