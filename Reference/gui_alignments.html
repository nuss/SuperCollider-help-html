    Alignment

    Alignment options within the GUI system

    Alignment in GUI classes is represented with the following symbols:

    • \left
    • \center
    • \right

    Additionally, the following symbols are available in Qt GUI:

    • \topLeft
    • \top
    • \topRight
    • \bottomLeft
    • \bottom
    • \bottomRight
