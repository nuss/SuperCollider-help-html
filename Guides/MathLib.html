                        MathLib
                        Extension

                        Some mathematical extensions to SuperCollider
                         

                        MathLib is a diverse library of mathematical functions.

                        Components

                        Cartesian, Spherical & Quaternion

                        Supported by Cartesian &c.

                        L-system

                        L-system support is offered by -rewriteString.

                        LaTeX

                        LaTeX supports integration with the LaTeX typesetting system.

                        Markov & Fuzzy Sets

                        Supported for Markov and Fuzzy Sets is provided here: Markov & Fuzzy.

                        Matrix

                        Matrix operations are provided by Matrix.

                        Patterns

                        Various Pattern operations are supported by Patterns.

                        Polynomials & Special Functions

                        Support for polynomials is provided here: Polynomials & Special.

                        Scatter Views: 2D & 3D Views

                        2D & 3D views are supported by ScatterViews.

                        Solvers

                        Support for numerical solving and root finding is found here: Solvers.

                        Statistics

                        Additional statistics methods for arrays offers a guide to various operations.
