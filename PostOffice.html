    PostOffice
    Extension

    customize handling of post messages
    See also: Post, Log

    Description

    PostOffice redirects posting and allows: - keeping posted lines, and - adding a function to run for every post message.

    First code examples:

    Class Methods

    .lines

    the current list of stored post lines

    .clear

    clear list of stored post lines

    .postfunc

    a function to run when a post comes in

    .add

    add a string / post message to lines
