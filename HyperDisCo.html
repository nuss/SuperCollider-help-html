    HyperDisCo
    Extension

    A modular library for internet-networked music systems.
    See also: OSCRouterClient, SyncText, CodingDojo

    HyperDisCo provides:

    • OSCRouterClient, an interface for tcposcrouter by Bruno Gola, see https://github.com/bgola/tcposcrouter.
    • SyncText for syncing code/text documents for teaching, live coding and performance
    • CodingDojo for holding CodingDojo sessions via internet
    • Workshop Materials:

    see also Quark("Utopia")

    HyperDisCo setup and Workshop

    To create an example HyperDisCo setup as used for a workshop, please do:

    1. Open the HyperDisCo/examples folder:

    (Quark("HyperDisCo").localPath +/+ "examples").openOS;

    2. Copy the "hyperfolder" to a convenient place, e.g. the Desktop

    3. Copy the "hyperdisco_loadMe.scd" file to a convenient location, e.g. to the Desktop, or if you use the StartupFile quark, to

    StartupFile.filesDir.openOS;

    4. Personalize your copy of the hyperdisco_loadMe file by following its instructions

    5. Run the entire file to load a full demo setup.

    6. Walk through the files in your copy of the "hyperfolder" and enjoy!
