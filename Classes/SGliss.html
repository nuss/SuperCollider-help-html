      SGliss : AbstractSInstrument : Object
      Extension

      a glissing sInstrument
      Source: SGliss.sc
      See also: sInstrumentsOverview, SGlissAmplitudes, SGlissFrequencies, SGlissPannings

      Description

      Written together with Tamas Ungvary.

      See file instruments/sGliss.scd for a complete example.

      Class Methods

      SGliss.new(maxNumVoices: 100, target, outbus: 0, addAction: 'addToHead', args)

      create a new gliss instrument. the localhost server will be booted if it is not already running.

      Arguments:

      maxNumVoices

      the maximum number of voices one can add to this instrument.

      target

      if nil then the localhost server's default group will be used.

      outbus

      where to send the stereo output.

      addAction

      a Symbol.

      args

      extra arguments as an Array with key/value pairs.

      Inherited class methods

      Instance Methods

      .voices

      the currently active voices.

      Returns:

      a List with AbstractSGlissVoice instances.

      .release(releaseTime: 1)

      fade out the sound and free all internal nodes and busses.

      Arguments:

      releaseTime

      fade out time in seconds.

      .set( ... args)

      set parameters for all internal nodes and save them in the arguments dictionary.

      Arguments:

      ... args

      an Array with key/value pairs. can also be a single key/value pair.

      .setPannings(arr)

      convenience method for .set(\pannings, ...)

      Arguments:

      arr

      an array of floats (-1.0 to 1.0).

      .getPannings

      read the panning positions control rate bus synchronously.

      Returns:

      an array of floats (-1.0 to 1.0).

      .setAmplitudes(arr)

      convenience method for .set(\amplitudes, ...)

      Arguments:

      arr

      an array of floats (0.0 to 1.0).

      .getAmplitudes

      read the amplitude control rate bus synchronously.

      Returns:

      an array of floats (0.0 to 1.0).

      .setFrequencies(arr)

      convenience method for .set(\frequencies, ...)

      Arguments:

      arr

      an array of floats in Hz.

      .getFrequencies

      read the frequencies control rate bus synchronously.

      Returns:

      an array of floats in Hz.

      .numVoices

      the current number of voices.

      .addVoices(num: 1, type: 'Sine', args)

      create new voices up to a maximum of maxNumVoices (set in *new). internally controllers for frequencies, amplitudes and panning are automatically created.

      Arguments:

      num

      the number of voices to be added.

      type

      a Symbol defining the type of sounds to add. valid types are \Saw, \Sine, \SineFB, \Pulse, \VOsc, \Formant, \Noise, \Pink, \File, \Folder.

      args

      extra arguments as an Array with key/value pairs.

      .removeVoices(num: 1, action)

      remove voices.

      Arguments:

      num

      the number of voices to be removed.

      action

      a Function that will be evaluated when the voice has faded out.
