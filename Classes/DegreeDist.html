    DegreeDist : Object
    Extension

    Higher Order Ambisonic (HOA) "by degree" distance radial filter
    Source: HoaUGen.sc
    See also: HoaNFDist, DegreeProx, DegreeCtrl

    Description

    Higher Order Ambisonic signal (HOA) near-field effect (NFE), distance radial filter by Associated Legendre degree (ℓ).

    NOTE: As a utility, DegreeDist supports NFE radial filtering and transformation. In most cases users will not directly use this filter. Instead, HoaNFDist profides the near-field effect distance transform operation on input HOA signals.

    Class Methods

    DegreeDist.ar(in, radius, degree: 0)

    Arguments:

    in

    The input signal.

    radius

    The decoding radius in meters.

    degree

    Associated Legendre degree (ℓ)

    Discussion:

    DegreeDist is implemented as a cascade of SOS and FOS filters, where coefficients are calculated by NFECoeffs: -dist.
