      Description

      MIDIAnalysis is an unfinished sketch for more detailed analysis of MIDI events as collected by MIDIExplorer or MIDIMonitor. May be moved to MIDIMonitor eventually.

      Class Methods

      *analyse (rawCapture)

      analyse a captured list of midi events.

      *analyseNoteEls

      analyse the noteOn/off events in the list of midi events.
