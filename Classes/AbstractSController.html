    AbstractSController : Object
    Extension

    an abstract class for all sInstrument controllers
    Source: AbstractSController.sc
    Subclasses: SClusterAmplitudes, SClusterDistortions, SClusterFrequencies, SDroneAmplitudes, SDroneFrequencies, SFormantAmplitudes, SFormantFrequencies, SGlissAmplitudes, SGlissFrequencies, SGlissPannings… see all
    See also: sInstrumentsOverview, AbstractSInstrument

    Description

    Multi-channel control rate bus+synth for amplitudes, frequencies, panning positions etc.

    NOTE: do not instantiate. See subclasses for examples of how to inherit.
    • AbstractSController - an abstract class for all sInstrument controllers
      • SClusterAmplitudes - control SCluster amplitudes
      • SClusterDistortions - control SCluster distortions
      • SClusterFrequencies - control SCluster frequencies
      • SDroneAmplitudes - control SDrone amplitudes
      • SDroneFrequencies - control SDrone frequencies
      • SFormantAmplitudes - control SFormant amplitudes
      • SFormantFrequencies - control SFormant frequencies
      • SGlissAmplitudes - control SGliss amplitudes
      • SGlissFrequencies - control SGliss frequencies
      • SGlissPannings - control SGliss panning positions

    Class Methods

    AbstractSController.new(numChannels: 5, target, args)

    Arguments:

    numChannels

    number of internal control rate channels.

    target

    used to figure out which server to use for the internal bus and synth. if nil then the default (localhost) server will be used.

    args

    extra arguments as an Array with key/value pairs.
