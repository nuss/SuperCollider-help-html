      NdefMixerOld

      mix control for all Ndefs on a given server
      Location: NOT INSTALLED!
      See also: NdefMixer

      Description

      For more details see: ProxyMixer

      Class Methods

      Creation

      .new

      Return a new window for a given server.

      // ndef mxers for other servers
n = NdefMixer(\internal);
n = NdefMixer(\localhost);

      Arguments:

      Server object ( Server ) or server name ( Symbol )

      an Integer.
