    RedGIFControl : Object
    Extension

    helper class for RedGIF
    Source: RedGIF.sc
    See also: RedGIF, RedGIFImage

    Description

    Holds data for specific images.

    NOTE: Do not instantiate. It is only a helper class for RedGIF.
