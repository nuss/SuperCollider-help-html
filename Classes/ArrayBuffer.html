    ArrayBuffer : BufferProxy : Object
    Extension

    given an array, will allocate and fill a Buffer for use in Patch etc
    Source: Sample.sc

    Description

    Passes an array into a Patch for use in UGens which need an array supplied as a buffer index. If saved as a compile string to disk, saves the array with it.
