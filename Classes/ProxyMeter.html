      ProxyMeter : Object
      Extension

      easy metering of audio proxy levels
      Source: ProxyMeter.sc
      See also: Ndef, NodeProxy, ProxySpace, PrePostView

      Description

      ProxyMeter measures the peak levels of proxies (e.g. those showing on a proxymixer), determines pre- and post-slider volume, and can displays these two levels on a mixer, or generally, on a MonitorGui.

      Written by adc, based on suggestions of / discussions with Ricardo Guerreiro. Reuses ideas and code from AudioMeter by A. Bartetzki.

      (
Server.default = s = Server.local;
s.makeWindow;
s.boot;
p = p ?? { ProxySpace.new }; p.push;

m = ProxyMixer(p, 8);
)
    // make some proxies
(
~a = { Ringz.ar(Dust.ar([1, 1]), [600, 800], 0.05) };
~b = { Dust.ar([20, 80, 320]) * 0.5 };
~y = { SinOsc.ar(0.2).abs * PinkNoise.ar };
~x = { SinOsc.ar(0.13).abs * BrownNoise.ar };

~a.play(vol: 0.2);
~b.play(vol: 0.1);
~x.play(vol: 0.2);
~y.play(vol: 0.1);
)
    // show pre- and post volume amps on the sliders
ProxyMeter.addMixer(m);

    // see CLIPPING WARNING - CAREFUL, THIS IS LOUD!
    // when level of ~x exceeds +-1, this would create
    // digital clipping when s.volume.volume is full on.
    // so turn down master for safety
s.volume.volume = -12;
    // now watch nodeproxy level going up to +2 go red:
~x = { SinOsc.ar(0.13).abs * BrownNoise.ar * 2 };
    // when when post-monitor level also exceeds 1, deep red:
~x.vol_(0.7);

    // where the proxy meter instances live
ProxyMeter.all;
ProxyMeter.showKrs; // show them as kr proxies
ProxyMeter.hideKrs;

    // remove them again
ProxyMeter.removeMixer(m);
