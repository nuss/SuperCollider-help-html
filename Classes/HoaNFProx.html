      HoaNFProx : HoaUGen : Object
      Extension

      Higher Order Ambisonic (HOA) NFE proximity transformer
      Source: HoaUGen.sc
      See also: HoaNFDist, HoaNFCtrl, DegreeDist, DegreeProx, DegreeCtrl

      Description

      Higher Order Ambisonic signal (HOA) near-field effect (NFE) proximity transformer.

      Class Methods

      HoaNFProx.ar(in, order)

      Arguments:

      in

      The input signal.

      order

      Ambisonic order.

      Discussion:

      HoaNFProx translates an input HOA signal where refRadius = AtkHoa.refRadius to refRadius = inf.

      This transform offers an HOA equivalent to FoaProximity where distance = AtkHoa.refRadius, in the first order case.

      NOTE: This operation may be used to decode signals encoded as HOA with NFE to FOA or third party Ambisonic signal sets.
      WARNING: The near-field effect proximity transformer is inherently numerically unstable and requires the use of signal pre-conditioned input to avoid overflow. In many cases, the use of HoaNFCtrl is advised, instead.
