      DWGReverbC1C3_16 : MultiOutUGen : UGen : AbstractFunction : Object
      Extension

      Digital waveguide reverberator
      Source: DWGReverb.sc
      See also: DWGReverbC1C3, DWGReverb3Band, DWGReverb3Band_16, EarlyRefGen, EarlyRefAtkGen, GVerb

      Description

      The same as DWGReverbC1C3 but with 16 feedback delays instead of 8 (best reverb coloration for short lengths) A mono-to-stereo reverb that models late reflections using a feedback delay network. Use in combination with EarlyRefGen or EarlyRefAtkGen for best results.

      in

      len

      Max size of the delay lines.

      c1

      Inverse of decay time.

      c3

      Inverse of height frequency decay time.

      mix

      Wet and dry mix factor

      coefs

      size len factor for every delay line

      perm
