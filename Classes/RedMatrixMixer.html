      RedMatrixMixer : Object
      Extension

      mix any number of input and output channels
      Source: RedMatrixMixer.sc
      See also: RedMatrixMixerGUI, RedEffectsRack, RedMixer

      Description

      NOTE: Requires the Conductor quark.

      Class Methods

      RedMatrixMixer.new(nIn: 8, nOut: 8, in: 0, out: 0, group, lag: 0.05)

      Arguments:

      nIn

      Number of input busses.

      nOut

      Number of output busses.

      in

      Offset of first input bus.

      out

      Offset of first out bus.

      group

      nil boots the default server and creates a group after the defaultGroup.

      lag

      For amplitudes in seconds.

      Inherited class methods

      Instance Methods

      Revert all CVs to their defaults.

      .free

      Free all synths.

      .gui(position)

      Create a RedMatrixMixerGUI for this RedMatrixMixer.

      Arguments:

      position

      A Point.

      .group

      The Group in use.

      .cvs

      Dictionary of CVs:

      ininput bus offset
      outoutput bus offset
      lagmix lag time in seconds (linear)
      o0, o1... oNas many as out busses. An array of amplitudes (as many as in busses)

      .synth

      The mixer synth.

      Examples

      a= RedMatrixMixer(4, 2, 0, 0);    //4 channels in, 2 channels out
(
SynthDef(\redMatrixMixerTest, {|out= 0, freq= 400, gate= 1, amp= 0.5|
    var e= EnvGen.kr(Env.asr(0.01, 1, 0.01), gate, doneAction:2);
    var z= SinOsc.ar(freq, 0, amp);
    Out.ar(out, z*e);
}).add;
)
(
b= Pbind(
    \instrument, \redMatrixMixerTest,
    \degree, Pseq([0, 1, 5, 4, 3, 4], inf),
    \dur, 0.25,
    \amp, 0.7,
    \out, Pseq([0, 1, 2, 3], inf)
).play;
)

a.gui;
a.cvs.o0.value= [1, 0, 0.3, 0];        //in0 to out0 with full amp, in2 to out0 with 0.3 amp
a.cvs.o1.value= [0, 1, 0, 0.2];        //in1 to out1 with full amp, in3 to out1 with 0.2 amp
a.cvs.in.value= 1;            //change to listen to input busses to 1-4

a.cvs.out.value= 1;            //left channel still outputs because the pbinds plays there
                    //and no ReplaceOut is blocking it
a.cvs.out.value= 0;
a.cvs.in.value= 0;

a.cvs.lag.value= 3;
a.cvs.o0.value= [0, 1, 1, 0.3];        //change mix with a 3 second lag
a.cvs.o1.value= [1, 0, 0, 0.2];
a.os;

a.defString
a.def

a.defaults
a.free;
b.stop;
