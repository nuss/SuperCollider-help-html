      InterplXYC : Object

      envelope specification
      Source: InterplEnv.sc
      See also: InterplEnv

      Description

      Takes sets of x, y and curve values and returns a new instance of InterplEnv. x values can be negative (for use in indexing with negative values or signals). See InterplEnv Help for more info.
