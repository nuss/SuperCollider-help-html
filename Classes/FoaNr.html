      FoaNr : FoaEval : FoaUGen : Object
      Extension

      First Order Ambisonic (FOA) Soundfield Unit Normalized Reactive Intensity Vector
      Source: ATKAnalyze.sc
      See also: FoaMagNa, FoaMagNr, FoaNa

      Description

      Return the Unit Normalized Reactive Intensity Vector, Nr, of a first order ambisonic (B-format) signal.

      NOTE: FoaNr is usually called via the convenience wrapper FoaAnalyze.

      Class Methods

      FoaNr.ar(in, size: 2048, method: 'instant')

      Arguments:

      in

      The B-format signal, an array: [w, x, y, z]

      size

      Analysis filter kernel size.

      NOTE: For method: 'instant', must be a power of 2.
      method

      The Soundfield analysis method:

      • 'instant'
      • 'average'
      WARNING: This UGen requires the reactive intensity vector, so will throw an error if called with the 'average' keyword.

      Please see discussion of method here.
