      AtkFoa : Object
      Extension

      A class that stores some global FOA variables for the Ambisonic Toolkit
      Source: AtkFoa.sc
      See also: Atk, AtkHoa, ATK Glossary

      Description

      The AtkFoa class defines variables that may be used to integrate FOA and HOA workflows within the ATK library.

      Class Methods

      Component ordering & normalisation

      AtkFoa.format

      Designation indicating ambisonic component ordering and component normalisation. The Ambisonic Toolkit FOA convention is FuMa-FuMa, i.e., Furse-Malham (FuMa), Gerzon / Furse-Malham (MaxN).

      AtkFoa.ordering

      Ambisonic Toolkit FOA ambisonic component ordering convention: Furse-Malham (FuMa). See ordering.

      AtkFoa.normalisation

      Ambisonic Toolkit ambisonic component normalisation convention: Gerzon / Furse-Malham (MaxN). See normalisation.

      Radial & NFE

      AtkFoa.refRadius

      Ambisonic Toolkit FOA reference radius, in meters.
