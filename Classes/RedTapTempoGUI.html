      Class Methods

      *new (clock, n: 4, timeout: 3, server, parent, position)

      Arguments:

      clock

      Provide a TempoClock. nil creates a TempoClock internally.

      n

      Number of taps to average.

      timeout

      Time after n counter is reset.

      server

      For monitor synth. The default is Server.default.

      parent

      A Window, CompositeView or nil.

      position

      A Point or nil.

      *def

      Return the SynthDef.

      Inherited class methods

      Get or set tempo in bps.

      -monitor = bool

      A Boolean.

      -monitorAmp = val

      A Float.

      -monitorBus = index

      Default monitor bus is 7.
