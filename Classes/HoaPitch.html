      HoaPitch : HoaTumble : HoaUGen : Object
      Extension

      Higher Order Ambisonic (HOA) rotation transformer
      Source: HoaUGen.sc
      See also: HoaRotate, HoaTilt, HoaTumble, HoaRTT, HoaYaw, HoaRoll, HoaYPR

      Description

      Rotate a Higher Order Ambisonic signal (HOA) around the y-axis.

      NOTE: HoaPitch is a synonym for HoaTumble

      Class Methods

      HoaPitch.ar(in, angle: 0, order)

      From superclass: HoaTumble

      Arguments:

      in

      The input signal.

      angle

      Rotation angle, in radians.

      order

      Ambisonic order.

      Discussion:

      A rotation of pi/2 will rotate a source at [0, 0] to [0, pi/2].
