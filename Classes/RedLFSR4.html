      RedLFSR4 : Object
      Extension

      4-bit linear feedback shift register
      Source: RedLFSR4.sc
      See also: RedLFSR4BitStream, RedNoise

      Description

      4-bit fibonacci LFSR. see https://en.wikipedia.org/wiki/Linear_feedback_shift_register

      Class Methods

      RedLFSR4.ar(trig: 0, iseed: 8)

      RedLFSR4.kr(trig: 0, iseed: 8)

      Arguments:

      trig

      Trigger that will calculate a new value.

      iseed

      Initial seed. Use 8 (2r1000) to get the common sequence: [4, 2, 9, 12, 6, 11, 5, 10, 13, 14, 15, 7, 3, 1, 8]

      Returns:

      Pseudo random numbers 0-15.
