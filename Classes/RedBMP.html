      RedBMP : Object
      Extension

      bitmap images
      Source: RedBMP.sc
      See also: RedGIF

      Description

      Read and write .bmp files from disk. Handles 1, 4, 8, 16, 24 and 32bit depths. Includes a simple -makeWindow method to display the pixels (but you should really write your own).

      See http://en.wikipedia.org/wiki/BMP_file_format

      Class Methods

      RedBMP.new(width: 320, height: 240, depth: 32)

      Create an empty bitmap image.

      RedBMP.read(path)

      Create a RedBMP by reading a .bmp file from disk.

      Instance Methods

      .write(path)

      Write the bitmap image to disk.

      .makeWindow(bounds)

      As simple as possible.

      .type

      A string. "BM".

      .width

      .width = value

      .height

      .height = value

      Image dimensions in pixels.

      .depth

      .depth = value

      Colour depth. 1, 4, 8, 16, 24 or 32.

      .compression

      When read from disk.

      .palette

      .palette = value

      Colour table as a 1-dimensional array of Color objects. Size 256 for 8bit images, size 16 for 4bit images and size 2 for 1bit images.

      .data

      .data = value

      The pixel data as a 1-dimensional array of Color objects. Reading from left to right, top to bottom. If you create your own make the size width*height.
