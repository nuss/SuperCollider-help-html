      See also: HoaNFProx, HoaNFCtrl, DegreeDist, DegreeProx, DegreeCtrl

      Description

      Higher Order Ambisonic signal (HOA) near-field effect (NFE) distance transformer.

      Class Methods

      HoaNFDist.ar(in, order)

      Arguments:

      in

      The input signal.

      order

      Ambisonic order.

      Discussion:

      HoaNFDist translates an input HOA signal where refRadius = inf to refRadius = AtkHoa.refRadius.

      This transform offers an HOA equivalent to FoaNFC where distance = AtkHoa.refRadius, in the first order case.

      NOTE: This operation is required to import FOA or third party signals into the ATK's HOA with NFE signal set format.
