    Class Methods

    AbstractSInstrument.new(numChannels: 2, target, outbus: 0, addAction: 'addToHead', args)

    Creates a new instrument. Will boot the localhost server if it is not already running.

    Arguments:

    numChannels

    number of output audio channels.

    target

    if nil then the localhost server's default group will be used.

    outbus

    stereo output.

    addAction

    a Symbol.

    args

    extra arguments as an Array with key/value pairs.
