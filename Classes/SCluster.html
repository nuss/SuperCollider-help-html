      Description

      See file instruments/sCluster.scd for a complete example.

      Class Methods

      SCluster.new(numVoices: 4, target, outbus: 0, addAction: 'addToHead', args)

      create a new cluster instrument. the server must be running before starting. there are only two channels (stereo)

      Arguments:

      numVoices

      the number of voices for this instrument.

      target

      if nil then the localhost server's default group will be used.

      outbus

      where to send the stereo output.

      addAction

      a Symbol.

      args

      extra arguments as an Array with key/value pairs.

      SCluster.type

      a Symbol describing the type of sound produced for example \SineFB.

      Instance Methods

      .defName

      a Symbol used for the internal SynthDef.

      .func

      a Function that will be wrapped inside the internal SynthDef.

      .release(releaseTime: 0.1, onFreeFunc)

      fade out the sound and free all internal nodes and busses.

      Arguments:

      releaseTime

      fade out time in seconds.

      onFreeFunc

      a Function to be evaluated after the reverberating sound disappeared.

      .xset( ... args)

      cross fade to a new set of voices. will also set parameters for all internal nodes and save them in the arguments dictionary. set the \atk and \rel parameters to define cross fade time (both are 4 by default).

      Arguments:

      ... args

      an Array with key/value pairs. can also be a single key/value pair.

      .set( ... args)

      set parameters for all internal nodes and save them in the arguments dictionary.

      Arguments:

      ... args

      an Array with key/value pairs. can also be a single key/value pair.
