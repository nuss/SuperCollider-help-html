    CVCenterShortcutsEditor : Object
    Extension

    An editor for CVCenter, CVWidget and CVWidget(MS)Editor shortcuts
    Source: CVCenterShortcutsEditor.sc
    See also: CVCenter, CVWidget, CVWidgetKnob, CVWidget2D, CVWidgetMS, AbstractCVWidgetEditor, CVWidgetEditor, CVWidgetMSEditor

    Description

    Shortcuts edited within this editor get effective immediately but last only until next recompile (except the option 'save to preferences' is checked).

    The dialog-window contains a number of tabs: CVCenter, CVWidget, CVWidget(MS)Editor (AbstractCVWidgetEditor) and global shortcuts. Each one of these holds a KeyDownActionsEditor.
