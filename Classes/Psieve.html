    Description

    PSVx sieve patterns inherit from Psieve. Normally you don't have to use Psieve directly. For an introduction and examples see Sieves and Psieve patterns.

    Class Methods

    Psieve.limit

    Psieve.limit = value

    Getter and setter for Integer limit of all Psieve patterns. You might want to set it globally when working with large numbers. Default to 65536.
