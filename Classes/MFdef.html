    MFdef : MFunc : AbstractFunction : Object
    Extension

    a named MFunc, a proxy for multiple modal functions
    Source: MFunc.sc
    See also: MFunc

    Description

    MFdef is a named MFunc, i.e. a named proxy for multiple modal functions. As with all the *def classes, instances can be looked up and created by key, and are accessible in a global dict called *all.

    Class Methods
