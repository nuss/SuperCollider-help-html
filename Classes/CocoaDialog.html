        Description

        OSX only. See GUI for cross-platform information.

        Class Methods

        *getPaths

        Displays an Open File Dialog.

        Arguments:

        okFunc

        A Function to be evaluated when OK is pressed. The selected paths are passed as an Array of Strings as the first argument.

        cancelFunc

        A Function to be evaluated if Cancel is pressed.

        allowsMultiple

        A Boolean indicating whether the dialog allows the user to select multiple files. The default is true.

        *savePanel

        Displays a Save File Dialog.

        Arguments:

        okFunc

        A Function to be evaluated when OK is pressed. The selected path will be passed as a String as the first argument.

        cancelFunc

        A Function to be evaluated if Cancel is pressed.
