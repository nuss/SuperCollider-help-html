      RedComb : Object
      Extension

      comb delay line with crossfade
      Source: RedComb.sc
      See also: RedDelay

      Description

      When delaytime is changed, a quick crossfade is performed.

      NOTE: This pseudo-ugen uses 2 delays internally and will allocate double amount of memory.
