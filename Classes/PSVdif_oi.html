      PSVdif_oi : Psieve : Pattern : AbstractFunction : Object
      Extension

      Sieve pattern for difference of integer generators with offsets and interval output
      Source: Psieve.sc
      See also: miSCellaneous, Sieves and Psieve patterns, Sieve, Psieve, PSVunion, PSVunion_i, PSVunion_o, PSVunion_oi, PSVsect, PSVsect_i, PSVsect_o, PSVsect_oi, PSVsymdif, PSVsymdif_i, PSVsymdif_o, PSVsymdif_oi, PSVdif, PSVdif_i, PSVop, PSVop_i, PSVop_o, PSVop_oi

      Description

      Pattern for difference of integer generators with offsets and interval output. Corresponds to Sieve's methods Sieve: *dif_oi and Sieve: -dif_oi. For an introduction and more examples see Sieves and Psieve patterns.

      Class Methods

      PSVdif_oi.new(genList, maxLength: inf, limit)

      Creates a new PSVdif_oi object.

      Arguments:

      genList

      An array of generators and corresponding offsets. Allowed generators: Integers, Streams or Patterns producing intervals or Sieves itself. Integers and Stream/Pattern output must be strictly positive. Integers as generators produce constant intervals. Offsets must be integers.

      maxLength

      Integer. Maximum number of items, which the stream will return. Defaults to inf.

      limit

      Integer. Limit up to which intervals can be returned by the stream. If no limit is passed, integer intervals might be returned up to default summation limit of 65536.
