      Description

      Providing help for accessing/communicating with an Ableton Push. Currently implements an interface to conveniently set colors.

      Class Methods

      *buttonLightCode (intensity: 'full', blink: 'steady', color: 'red', row: 0)

      returns a cc-number representing the color specified by given arguments.

      Arguments:

      intensity

      can be either \dim, \full or \half (2nd row only)

      blink

      can be either \steady, \slow or \fast (not yet implemented for 2nd row)

      color

      A Symbol representing the color.

      MPush.lightDict.topRowColor.keys; // 1st row
MPush.lightDict.padColor.keys; // 2nd row
      row

      either 0 or 1.

      *padLightCode (intensity: 'full', blink: 'steady', color: 'red')

      returns a velocity representing the color specified by the given arguments.

      Arguments:

      intensity

      can be either \dim, \full or \half

      blink

      can be either \steady, \slow or \fast (not yet implemented)

      color

      A Symbol representing the color.

      MPush.lightDict.padColor.keys;
      (row)

      The row of the pads
