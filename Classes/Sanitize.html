    Description

    Replaces infinities, NaNs, and subnormal numbers (denormals) with a given signal, zero by default. The method UGen: -sanitize provides a shorthand for this.

    See also CheckBadValues, which allows you to discriminate specific kinds of bad values and print information about them to the post window.
