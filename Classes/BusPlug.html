        Description

        BusPlug is mainly in use as a basic superclass of NodeProxy, but it can be applied for other things as well. Most methods are documented in the NodeProxy helpfile.

        Instance Methods

        -clear

        Free the bus, end the monitor

        -ar (numChannels, offset: 0)

        -kr (numChannels, offset: 0)

        Return a link to my output, which is limited by numChannels. If uninitialized, creates a matching bus. Normally, ar defaults to stereo, kr to mono. This can be set in the classvars: *defaultNumAudio, *defaultNumControl

        -play (out, numChannels, group, multi: false, vol, fadeTime, addAction)

        Plays from a bus index ( out ) with a number of channels to another index with a number of channels, within a group (ie a target group or server).

        Arguments:

        out

        bus index

        numChannels

        an Integer

        group

        target Group or Server

        multi

        keep old links and add new one

        vol

        volume at which to monitor

        fadeTime

        fade in fade out time

        addAction

        -playN (outs, amps, ins, vol, fadeTime, group, addAction)

        Examples

        // using as a control bus listener

s.boot;
z = Bus.control(s, 16);
a = BusPlug.for(z);

m = { Mix(SinOsc.ar(a.kr(16), 0, 0.1)) }.play;

z.setn(Array.rand(16, 300, 320).put(16.rand, rrand(500, 1000)));
z.setn(Array.rand(16, 300, 320).put(16.rand, rrand(500, 1000)));
z.setn(Array.rand(16, 300, 320).put(16.rand, rrand(500, 1000)));

m.free;


m = { SinOsc.ar(a.kr(2, MouseX.kr(0, 19)), 0, 0.1) }.play; // modulate channel offset

z.setn(Array.rand(16, 300, 1320).put(16.rand, rrand(500, 1000)));


m.free; z.free;

// using as a audio monitor

p = BusPlug.audio(s,2);
d = { Out.ar(p.index, PinkNoise.ar([0.1, 0.1])) }.play;


p.play; // monitor whatever plays in p (the execution order does not matter)



d.free;
d = { Out.ar(p.index, PinkNoise.ar([0.1, 0.1])) }.play;

p.stop;
p.play;

// also p can play to another bus:

p.stop;
p.play(12);

// listen to that bus for a test:
x = { InFeedback.ar(12,2) }.play;
x.free;
