      RedEffectsRackGUI : Object
      Extension

      gui for RedEffectsRack
      Source: RedEffectsRackGUI.sc
      See also: RedEffectsRack, RedMatrixMixerGUI, RedMixerGUI

      Description

      NOTE: Requires the Conductor and the TabbedView quarks.

      Class Methods

      RedEffectsRackGUI.new(redEffectsRack, position)

      Arguments:

      redEffectsRack

      A RedEffectsRack object.

      position

      A Point.

      RedEffectsRackGUI.numEffectsBeforeScroll

      RedEffectsRackGUI.numEffectsBeforeScroll = value

      The number of RedEffectModuleGUI to fit before adding a vertical scroller. The default is 9. This fits well with a screensize of 1440x900. For smaller screens use a lower value.

      a= RedEffectsRack();
