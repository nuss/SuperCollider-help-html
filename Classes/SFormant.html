      SFormant : AbstractSInstrument : Object
      Extension

      a formant sInstrument
      Source: SFormant.sc
      See also: sInstrumentsOverview, SFormantAmplitudes, SFormantFrequencies

      Description

      Based on optoformant3.

      See file instruments/sFormant.scd for a complete example.

      Class Methods

      SFormant.new(target, outbus: 0, addAction: 'addToHead', args)

      create a new formant instrument. the server must be running before starting. there are only two channels (stereo)

      Arguments:

      target

      if nil then the localhost server's default group will be used.

      outbus

      where to send the stereo output.

      addAction

      a Symbol.

      args

      extra arguments as an Array with key/value pairs.

      Inherited class methods

      Instance Methods

      .release(releaseTime: 0.1, onFreeFunc)

      fade out the sound and free all internal nodes and busses.

      Arguments:

      releaseTime

      fade out time in seconds.

      onFreeFunc

      a Function to be evaluated after the reverberating sound disappeared.

      .xset( ... args)

      cross fade to a new set of voices. will also set parameters for all internal nodes and save them in the arguments dictionary. set the \atk and \rel parameters to define cross fade time (both are 4 by default).

      Arguments:

      ... args

      an Array with key/value pairs. can also be a single key/value pair.

      .set( ... args)

      set parameters for all internal nodes and save them in the arguments dictionary.

      Arguments:

      ... args

      an Array with key/value pairs. can also be a single key/value pair.

      .setAmplitudes(arr)

      convenience method for .set(\amplitudes, ...)

      Arguments:

      arr

      an array of floats (0.0 to 1.0).

      .getAmplitudes

      read the amplitude control rate bus synchronously.

      Returns:

      an array of floats (0.0 to 1.0).

      .setFrequencies(arr)

      convenience method for .set(\frequencies, ...)

      Arguments:

      arr

      an array of floats in Hz.

      .xsetFrequencies(arr)

      convenience method for .xset(\frequencies, ...)will crossfade.

      Arguments:

      arr

      an array of floats in Hz.

      .getFrequencies

      read the frequencies control rate bus synchronously.

      Returns:

      an array of floats in Hz.
