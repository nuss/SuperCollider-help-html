      RedBitStream2 : RedBitStream : Stream : AbstractFunction : Object
      Extension

      stream bits
      Source: RedBitStream.sc
      See also: CollStream, RedBitStream

      Description

      Modeled after CollStream. Produces a bit stream out if an array of integers. The most significant bit comes first in each byte (MSB). Use class RedBitStream for least significant bit first (LSB).

      Class Methods

      RedBitStream2.new(collection, byte: 8)

      From superclass: RedBitStream

      Arguments:

      collection

      An Array of integers.

      byte

      Number of bits per byte in this collection of integers. If byte is 8 then integers should not be greater than 255.
