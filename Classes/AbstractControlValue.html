    AbstractControlValue : Object
    Extension

    Base class for control values.
    Source: ControlValue.sc
    Subclasses: IndexedControlValue, NumericControlValue, OnOffControlValue
    See also: NumericControlValue, Connection overview

    Description

    This is an abstract base class for values which can be constained, and update dependants when they change. See subclasses for usage.
