      HoaEncodeDirection : HoaUGen : Object
      Extension

      Higher Order Ambisonic (HOA) panner
      Source: HoaUGen.sc
      See also: HoaEncodeMatrix, HoaDecodeDirection

      Description

      Encode a monophonic input to a Higher Order Ambisonic signal (HOA), with near-field effect (NFE).

      Class Methods

      HoaEncodeDirection.ar(in, theta: 0, phi: 0, radius, order)

      Arguments:

      in

      The input signal.

      theta

      Azimuth, in radians.

      phi

      Elevation, in radians.

      radius

      Radius, in meters.

      order

      Ambisonic order.

      Discussion:

      HoaEncodeDirection offers HOA encoding equivalent to FoaPanB followed by FoaProximity, in the first order case.
