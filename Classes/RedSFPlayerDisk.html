      RedSFPlayerDisk : RedSFPlayer : Object
      Extension

      plays soundfiles (from disk)
      Source: RedSFPlayerDisk.sc
      See also: RedSFPlayer

      Description

      Based on VDiskIn. Use RedSFPlayer for playing back soundfiles from memory instead.

      Class Methods

      RedSFPlayerDisk.new(server)

      From superclass: RedSFPlayer

      Create a new player on a Server.

      Arguments:

      server

      If nil it will use Server.default.

      RedSFPlayerDisk.read(path, server)

      From superclass: RedSFPlayer

      Create a new player on a Server.

      Arguments:

      path

      A String pointing to a soundfile.

      server

      If nil it will use Server.default.

      .loop(out: 0, rate: 1, fadeTime: 0)

      From superclass: RedSFPlayer

      .play(out: 0, rate: 1, fadeTime: 0)

      From superclass: RedSFPlayer

      Starts playing the soundfile.

      Arguments:

      out

      Output bus.

      rate

      Playback rate.

      NOTE: For this disk player version rates can not be negative.
      fadeTime

      In seconds.

      NOTE: For this disk player version there is no crossfade of loops.

      .stop(fadeTime: 0)

      From superclass: RedSFPlayer

      Arguments:

      Examples

      s.boot;
a= RedSFPlayerDisk.new
a.read(Platform.resourceDir +/+ "sounds/a11wlk01-44_1.aiff")
a.play                //default is play out on bus 0 with rate 1 and 0 fadeTime
a.stop

a.loop(0, 3, 4)        //start looping with long fadeTime and high rate
a.loop(0, 1.2)        //another loop (crossfade will not work and fadeTime is ignored)
a.stop(4)                //fadeout

a.free
